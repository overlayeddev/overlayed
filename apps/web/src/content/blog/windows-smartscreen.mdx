## ðŸ¤“ Technical mumbo jumbo

<Link
  target="_blank"
  to="https://learn.microsoft.com/en-us/windows/security/operating-system-security/virus-and-threat-protection/microsoft-defender-smartscreen/"
>Windows Defender SmartScreen</Link>
protects against phishing or malware websites and applications, and the
downloading of potentially malicious files.

Overlayed signs its windows executables (.msi and .exe) with a code signing certificate from <Link to="https://ssl.com">ssl.com</Link> in order to build trust with Microsoft. However, since Overlayed uses an <Link to="https://www.ssl.com/faqs/which-code-signing-certificate-do-i-need-ev-ov/">OV Code Signing certificate</Link> it again means trust has to be built over time.

You can read this <Link
  target="_blank"
  to="https://stackoverflow.com/a/66582477">stackoverflow post</Link> to get a better understanding of how it all works.
